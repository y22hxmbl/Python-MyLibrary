# MyLibrary

This is a School Project, it's not called MyLibrary because it has anything to do with a library, but because it's a project that we created to learn Python. 

I am going to have examples of code inside this one file as functions and should be able to call them from another function in the same file or a different file.

In order to run:
1. Clone the repository
2. Open the terminal and navigate to the project directory
3. Create a virtual environment using the command `python -m venv .venv` and activate it using `source .venv/bin/activate` (Linux/Mac) or `.venv\Scripts\activate.csh` (Windows CMD) or `.venv\Scripts\activate.ps1` (Windows PowerShell)
4. Install the required dependencies using the command `pip install -r requirements.txt`
5. Run the command `python main.py` to execute the main file which will call the
